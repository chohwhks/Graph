# Graph
# Демонстрация алгоритма поиска кратчайшего пути в графе


- Отображение графа в виде вершин и рёбер и правильная расстановка по кругу
- Динамический пересчёт веса рёбер как расстояния между вершинами
- Перетягивание вершин мышью с автоматическим пересчётом весов
- Поиск кратчайшего пути между двумя вершинами по алгоритму Дейкстры (реализация на std::priority_queue)
- Визуализация найденного пути анимированным движением "машинки" вдоль рёбер графа с поворотами вдоль пути

## Сборка
Открыть .pro-файл в Qt Creator, собрать через qmake.
